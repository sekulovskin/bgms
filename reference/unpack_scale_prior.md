# Unpack a scale prior into the flat parameters used by bgm_spec

Unpack a scale prior into the flat parameters used by bgm_spec

## Usage

``` r
unpack_scale_prior(prior)
```

## Arguments

- prior:

  A `bgms_scale_prior` object.

## Value

A list with `scale_prior_type` (character), `scale_shape` (numeric), and
`scale_rate` (numeric).
