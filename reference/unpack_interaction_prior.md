# Unpack an interaction prior into the flat parameters used by bgm_spec

Unpack an interaction prior into the flat parameters used by bgm_spec

## Usage

``` r
unpack_interaction_prior(prior)
```

## Arguments

- prior:

  A `bgms_parameter_prior` object.

## Value

A list with `interaction_prior_type` (character) and `pairwise_scale`
(numeric).
